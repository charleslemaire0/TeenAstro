Equinox Precession (equinox_precession.dart)

Converts equatorial coordinates from a catalog equinox (e.g. J2000) to JNow (apparent, equinox of date). Logic matches TeenAstro Ephemeris and LX200Navigation: the SHC converts catalog coordinates to JNow at UTC date before sending to the main unit. The main unit is always in JNow.

Mean Obliquity of the Ecliptic

Mean obliquity in arcseconds (IAU 1980):

ε₀ = 23°26'21.448" − 46.8150·T − 0.00059·T² + 0.001813·T³ where T = (JD − 2451545.0) / 36525 (Julian centuries from J2000)

Nutation (17-term series)

Mean longitudes (degrees):

Ls = 280.4565 + 36000.7698·T + 0.000303·T² (mean longitude of Sun) Lm = 218.3164 + 481267.8812·T − 0.001599·T² (mean longitude of Moon) Ms = 357.5291 + 35999.0503·T − 0.000154·T² (mean anomaly of Sun) Mm = 134.9634 + 477198.8675·T + 0.008721·T² (mean anomaly of Moon) ω = 125.0443 − 1934.1363·T + 0.008721·T² (longitude of ascending node of Moon)

Nutation in longitude Δψ (arcsec):

Δψ = −(17.1996 + 0.01742·T)·sin(ω) − (1.3187 + 0.00016·T)·sin(2Ls) − 0.2274·sin(2Lm) + 0.2062·sin(2ω) + (0.1426 − 0.00034·T)·sin(Ms) + 0.0712·sin(Mm) − (0.0517 − 0.00012·T)·sin(2Ls + Ms) − 0.0386·sin(2Lm − ω) − 0.0301·sin(2Lm + Mm) + 0.0217·sin(2Ls − Ms) − 0.0158·sin(2Ls − 2Lm + Mm) + 0.0129·sin(2Ls − ω) + 0.0123·sin(2Lm − Mm)

Nutation in obliquity Δε (arcsec):

Δε = (9.2025 + 0.00089·T)·cos(ω) + (0.5736 − 0.00031·T)·cos(2Ls) + 0.0977·cos(2Lm) − 0.0895·cos(2ω) + 0.0224·cos(2Ls + Ms) + 0.0200·cos(2Lm − ω) + 0.0129·cos(2Lm + Mm) − 0.0095·cos(2Ls − Ms) − 0.0070·cos(2Ls − ω)

True obliquity:

ε = ε₀ + Δε

Precession (annual)

Annual precession rates (seconds of time for RA, arcsec for Dec):

m = 3.07496 + 0.00186·Teq nRA = 1.33621 − 0.00057·Teq nDec = 20.0431 − 0.0085·Teq where Teq = equinox − 2000 ΔRA = m + nRA·sin(RA)·tan(Dec) ΔDec = nDec·cos(RA) RA += (ΔRA · yearDiff) / 3600 (convert sec of time → hours) Dec += (ΔDec · yearDiff) / 3600 (convert arcsec → degrees)

Nutation in RA/Dec

Convert nutation (Δψ, Δε in arcsec) to equatorial corrections:

ΔRAnut = (cos(ε) + sin(ε)·sin(RA)·tan(Dec))·Δψ − cos(RA)·tan(Dec)·Δε ΔDecnut = sin(ε)·cos(RA)·Δψ + sin(RA)·Δε

RA and Dec in the formulas are in degrees. Results are converted to hours (RA) and degrees (Dec).

Aberration

Constant of aberration: K = 20.49552"

Sun orbital elements:

L₀ = 280.46646 + 36000.76983·T + 0.0003032·T² (mean longitude) M = 357.5291092 + 35999.0502909·T − 0.0001536·T² (mean anomaly) e = 0.016708634 − 0.000042037·T − 0.0000001267·T² (eccentricity) π = 102.93735 + 1.71946·T + 0.00046·T² (longitude of perihelion)

Equation of center and true longitude:

C = (1.914602 − 0.004817·T − 0.000014·T²)·sin(M) + (0.019993 − 0.000101·T)·sin(2M) + 0.000289·sin(3M) ☉ (true longitude of Sun) = L₀ + C

Aberration corrections (arcsec → hours/degrees):

ΔRAab = −K·(cos(RA)·cos(☉)·cos(ε) + sin(RA)·sin(☉)) / cos(Dec) + K·e·(cos(RA)·cos(π)·cos(ε) + sin(RA)·sin(π)) / cos(Dec) ΔDecab = −K·(cos(☉)·cos(ε)·(tan(ε)·cos(Dec) − sin(RA)·sin(Dec)) + cos(RA)·sin(Dec)·sin(☉)) + K·e·(cos(π)·cos(ε)·(tan(ε)·cos(Dec) − sin(RA)·sin(Dec)) + cos(RA)·sin(Dec)·sin(π))

Planet Positions (planet_positions.dart)

Astronomical algorithms for Sun, Moon, and planet positions. Based on Jean Meeus Astronomical Algorithms (simplified, visual accuracy).

Julian Date

double julianDate(DateTime utc)

Meeus formula (Ch. 7):

If month ≤ 2: y = year − 1, m = month + 12 Else: y = year, m = month a = floor(y / 100) b = 2 − a + floor(a / 4) JD = floor(365.25·(y + 4716)) + floor(30.6001·(m + 1)) + day + b − 1524.5 where day = day + hour/24 + minute/1440 + second/86400

_jdCenturies(jd) = (jd − 2451545.0) / 36525.0

Mean Obliquity

meanObliquity(t) = 23.439291 − 0.0130042·t − 1.64×10⁻⁷·t² + 5.04×10⁻⁷·t³ (degrees, Meeus Ch. 22)

Ecliptic → Equatorial

(double ra, double dec) eclipticToEquatorial(double lonDeg, double latDeg, double obliquity)
sin(Dec) = sin(lat)·cos(ε) + cos(lat)·sin(ε)·sin(lon) RA = atan2(sin(lon)·cos(ε) − tan(lat)·sin(ε), cos(lon)) RA in hours = RA_deg / 15

Sun Position (Meeus Ch. 25)

Low accuracy ~1′:

L₀ = 280.46646 + 36000.76983·t + 0.0003032·t² (geometric mean longitude) M = 357.52911 + 35999.05029·t − 0.0001537·t² (mean anomaly) C (equation of center) = (1.914602 − 0.004817·t − 0.000014·t²)·sin(M) + (0.019993 − 0.000101·t)·sin(2M) + 0.000289·sin(3M) sunLon = L₀ + C ω = 125.04 − 1934.136·t apparentLon = sunLon − 0.00569 − 0.00478·sin(ω) ε = meanObliquity(t) + 0.00256·cos(ω) Then: eclipticToEquatorial(apparentLon, 0, ε)

Moon Position (Meeus Ch. 47)

Truncated ~20 terms, ~10′ accuracy. Mean arguments (degrees):

lp = 218.3164477 + 481267.88123421·t − 0.0015786·t² + t³/538841 (mean longitude) d = 297.8501921 + 445267.1114034·t − 0.0018819·t² + t³/545868 (mean elongation) m = 357.5291092 + 35999.0502909·t − 0.0001536·t² + t³/24490000 (mean anomaly Sun) mp = 134.9633964 + 477198.8675055·t + 0.0087414·t² + t³/69699 (mean anomaly Moon) f = 93.2720950 + 483202.0175233·t − 0.0036539·t² − t³/3526000 (argument of latitude)

Longitude terms (sumL, coefficients × 10⁻⁶):

6288774·sin(mp) 1274027·sin(2d−mp) 658314·sin(2d) 213618·sin(2mp) −185116·sin(m) −114332·sin(2f) 58793·sin(2d−2mp) 57066·sin(2d−m−mp) 53322·sin(2d+mp) 45758·sin(2d−m) −40923·sin(m−mp) −34720·sin(d) −30383·sin(m+mp) 15327·sin(2d−2f) −12528·sin(mp+2f) 10980·sin(mp−2f) 10675·sin(4d−mp) 10034·sin(3mp) 8548·sin(4d−2mp)

Latitude terms (sumB):

5128122·sin(f) 280602·sin(mp+f) 277693·sin(mp−f) 173237·sin(2d−f) 55413·sin(2d−mp+f) 46271·sin(2d−mp−f) 32573·sin(2d+f) 17198·sin(2mp+f) 9266·sin(2d+mp−f) 8822·sin(2mp−f) 8216·sin(2d−m−f) 4324·sin(2d−2mp−f)
lonMoon = lp + sumL / 1,000,000 latMoon = sumB / 1,000,000

Kepler Equation

double _solveKepler(double mDeg, double e)
M = E − e·sin(E) (mean anomaly M, eccentric anomaly E) Newton iteration: dE = (M − E + e·sin(E)) / (1 − e·cos(E)) E += dE (until |dE| < 1e-10, max 20 iterations)

Heliocentric Position (orbital plane)

xp = a·(cos(E) − e) yp = a·√(1 − e²)·sin(E)

Rotation to Ecliptic

Via Ω (ascending node), ω (perihelion), i (inclination):

argPeri = ω − Ω xecl = (cos(argPeri)·cos(Ω) − sin(argPeri)·sin(Ω)·cos(i))·xp + (−sin(argPeri)·cos(Ω) − cos(argPeri)·sin(Ω)·cos(i))·yp yecl = (cos(argPeri)·sin(Ω) + sin(argPeri)·cos(Ω)·cos(i))·xp + (−sin(argPeri)·sin(Ω) + cos(argPeri)·cos(Ω)·cos(i))·yp zecl = sin(argPeri)·sin(i)·xp + cos(argPeri)·sin(i)·yp

Geocentric

Subtract Earth position (same Kepler method). Geocentric ecliptic:

lonGeo = atan2(dy, dx) latGeo = asin(dz / dist) dist = √(dx² + dy² + dz²)

CelestialBody planetPosition(Planet planet, double jd)

Magnitude

_planetMag(planet, delta, r, e, mAnom) Approximate: mag = base + 5·log₁₀(δ·r) Mercury: −0.36 Venus: −4.34 Mars: −1.51 Jupiter: −9.40 Saturn: −8.88 Uranus: −7.19 Neptune: −6.87

Phase angle ignored for simplicity. delta = Earth–planet distance (AU), r = Sun–planet distance (AU).
