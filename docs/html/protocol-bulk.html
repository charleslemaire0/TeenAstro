Bulk Packets (GXAS / GXCS)

TeenAstro provides binary bulk packets for efficient state and config polling. Clients send :GXAS# or :GXCS# and receive base64-encoded little-endian binary data. A single packet replaces dozens of individual get commands.

GXAS Packet (102 bytes, 136 base64 chars)

Command: :GXAS#. Returns full mount state. Byte 101 = XOR checksum of bytes 0–100.

BytesTypeContent
0uint8Status: bits 0–1 track (0=off, 1=on, 2=slew); 2–3 sidereal; 4–5 park; 6 atHome; 7 pierSide
1uint8Status: bits 0–2 guidingRate; 3 aligned; 4–6 mountType; 7 spiralRunning
2uint8Status: bits 0–1 guidingEW; 2–3 guidingNS; 4–5 rateComp; 7 pulseGuiding
3uint8gnssFlags (hasBoard, valid, timeSync, locationSync, hdop)
4uint8error (MountState::Errors enum)
5uint8enableFlags (encoders, calibrating, pushTo, motors, focuser)
6–8uint8UTC: hour, minute, second
9–11uint8UTC date: month, day, year
12–19float64RA (hours)
20–27float64Dec (degrees)
28–35float64Alt (degrees)
36–43float64Az (degrees)
44–51float64LST (hours)
52–59float64Target RA
60–67float64Target Dec
68–75float64Current track rate RA
76–83float64Current track rate Dec
84–87int32Stored track rate RA
88–91int32Stored track rate Dec
92–95uint32Focuser position (steps)
96–97uint16Focuser speed
98int8Timezone offset (×10)
99uint8Alignment ref count (0–2)
100uint8Align phase (bits 0–1); align star num (bits 2–4)
101uint8XOR checksum of bytes 0–100

GXCS Packet (90 bytes, 120 base64 chars)

Command: :GXCS#. Returns full mount config. Byte 89 = XOR checksum of bytes 0–88.

BytesTypeContent
0–15axis 1Gear (u32), stepRot (u16), backlash (u16), backlashRate (u16), lowCurr (u16), highCurr (u16), micro (u8), flags (u8)
16–31axis 2Same layout as axis 1
32–35float32Guide rate
36–39float32Slow rate
40–43float32Medium rate
44–47float32Fast rate
48–51float32Acceleration (deg)
52–53uint16Max slew rate
54uint8Default rate index
55uint8Slew settle time (s)
56–59int16Meridian E/W limits
60–67int16User axis limits (min1, max1, min2, max2)
68–69uint16Under-pole limit
70–71int8Min/max altitude
72uint8Distance from pole
73uint8Refraction flags
74–83—Encoder config
84uint8Mount index
85–88—Reserved
89uint8XOR checksum of bytes 0–88

Focuser Packet (:Fa#)

Command: :Fa#. Returns focuser state. 12 base64 chars = 9 bytes.

BytesContent
0–3Position (steps)
4–5Speed
6–8Temperature (optional)

TeenAstroMountStatus C++ Class

TeenAstroMountStatus parses GXAS and GXCS into structured state. It caches the raw base64 and exposes MountState and config fields.

MountState Fields (from GXAS)

tracking, sidereal, parkState, atHome, guidingRate, aligned, alignmentRefCount, alignPhase, alignStarNum, mountType, pierSide, gnssFlags, error, enableFlags, ra, dec, alt, az, targetRa, targetDec, utcTime, utcDate, siderealTime, timezoneOffset, trackRateRa, trackRateDec, storedTrackRateRa, storedTrackRateDec, focuserPos, focuserSpeed, valid.

Enums

  • TrackState — off, on, slewing
  • SiderealMode — star, sun, moon, target
  • ParkState — unparked, parked, failed, parking
  • PierState — east, west
  • GuidingRate — guide, slow, medium, fast, max
  • Mount — GEM, Fork, AltAz, ForkAlt
  • Errors — none, motorFault, alt, limitSense, limitA1, limitA2, underPole, meridian, sync

StepperDriver Enum

StepDir, TOS100, TMC2130, TMC5160, etc. (from :GVb#).
