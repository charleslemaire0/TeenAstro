Audits & Safety

This page summarizes security and code-quality audits performed on the TeenAstro codebase: memory overflow fixes, build portability, app bugs, and orphan command removal.

Memory Overflow Audit Summary

SHC Fixes

Several buffer overflows were found in the SHC (catalog titles, constellation list, catalogSubMenu). Fixes applied: increased buffer sizes, strncpy/strncat with explicit bounds.

Webserver Safe

TeenAstroWifi HTTP form input is copied with strncpy and length checks. SSID, password, AP settings, and web password handling are bounded. No unchecked strcpy of server.arg() into fixed buffers found.

TeenAstroCatalog Fix

catalogSubMenu() had a definite overflow: Title[32] copied into thisSubMenu[16]. Fixed with strncpy(thisSubMenu, catalog[_selected].Title, sizeof(thisSubMenu)-1) and null termination.

Recommendations

  • Prefer bounded functions: Use strncpy/strncat/snprintf with explicit size for any buffer receiving variable-length or external data.
  • Centralize max lengths: For catalog titles/submenus, consider #define CATALOG_TITLE_MAX 32 and use it for both catalog_t.Title and local buffers.
  • Keep SHC_text.h checks: Existing static_assert checks prevent T_* strings from exceeding display and buffer sizes.
  • Constellation/object lists: If more catalogs or longer names are added, re-check string_list_fCon and string_list_fType sizes.

Build & App Audit

Portability

All required files are in the repo. No hidden “only on one PC” dependencies. Build scripts use generic paths; FlutterPath default is empty (rely on PATH); DevelopRoot uses $env:USERPROFILE.

TCP Disconnect Fix

File: teenastro_app/lib/services/lx200_tcp_client.dart. In disconnect(), the code previously cancelled the stream subscription first, then closed the socket. If onDone ran asynchronously, the socket might not be closed. Fix: Close socket first, then cancel subscription.

RA/Dec Formatting

.round() on fractional seconds could produce 60 (e.g. 59.7 → 60), giving “12:59:60”. Fix: Carry logic: if seconds ≥ 60 increment minutes; if minutes ≥ 60 increment hours/degrees.

Center-on-Object

Search-and-select set _rotation but not _panY, so object wasn’t centered vertically. Fix: Reset _panY = 0 when centering.

Star Size Clamp

Radius clamped to [0.5, 22.0] but cutoff was r < 0.3. Dead code; faintest stars inflated to 0.5px. Fix: Lower clamp minimum to 0.15 so faint-star cutoff works.

Constellation Fallback

build_app.ps1 fallback created { "constellations": [] } (JSON object) but parser expects a top-level array. Fix: Fallback now creates [].

Orphan Commands

Removed (GXJB..GXRf)

Commands :GXJB#, :GXJC#, :GXJm#, :GXJM1#, :GXJM2#, :GXJP#, :GXJS#, :GXJT#, :GXRr#, :GXRd#, :GXRh#, :GXRe#, :GXRf# returned data already in :GXAS#. They have been removed; clients use GXAS for state.

Kept (GR, GD, etc.)

Standard LX200 commands :GR#, :GD#, :GA#, :GZ#, :GS#, :Gr#, :Gd# are kept for protocol compatibility with planetarium software, simulators, and scripts.

The individual :GXMx#, :GXRx#, :GXLx# config getters remain available for config UIs that read or validate a single field; :GXCS# provides full config in one call.
