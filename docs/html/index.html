System Overview

TeenAstro is an open-source telescope mount controller designed for German Equatorial (GEM), Fork, Alt-Az, and hybrid mounts. The system comprises embedded firmware running on Teensy and ESP8266 microcontrollers, a multi-platform mobile/desktop application built with Flutter, and Windows-based ASCOM integration.

MainUnit

Teensy 3/4. Real-time control loop: motor drive, goto, tracking, multi-star alignment, encoder support.

SHC

ESP8266. Smart Hand Controller with OLED UI, button navigation, catalogs, menu system.

WiFi Server

ESP8266. TCP bridge (port 9999) exposing the LX200 serial protocol over WiFi.

Flutter App

Android/Windows. Dashboard, planetarium, catalog browser, goto, multi-star alignment wizard.

ASCOM V7

.NET C#. Windows ASCOM driver for Stellarium, Cartes du Ciel, and other planetarium software.

Focuser

Teensy. Dedicated electronic focuser with temperature sensing and serial passthrough.

User community: groups.io/g/TeenAstro

Communication Protocol

All communication uses the LX200 serial protocol: :CMD# format. The MainUnit is the protocol endpoint; the WiFi Server and SHC are transparent bridges/clients.

Real-Time Control Loop

  1. The MainUnit runs a sidereal clock at ~100 Hz (masterClockSpeed = 106, siderealClockSpeed ≈ 997269.5625).
  2. Timer ISRs (TIMER3, TIMER4) generate step pulses for each axis based on computed intervals.
  3. The main loop processes serial commands, updates tracking targets, runs the goto state machine, and performs safety checks.
  4. Clients poll state via the binary bulk packet :GXAS# (102 bytes, ≈10 Hz) and config via :GXCS# (90 bytes, on demand).

Coordinate Pipeline

The mount transforms coordinates through a well-defined pipeline:

Catalog (J2000 RA/Dec) │ precession + nutation + aberration ▼ Apparent (JNow RA/Dec) │ EQ → HO conversion (latitude, LST) ▼ Horizontal (Alt/Az) │ Atmospheric refraction (Saemundsson / Bennett) ▼ Apparent Horizontal │ Alignment matrix T (Taki + SVD) ▼ Instrument (Axis1, Axis2, Axis3) │ gear ratio, microstep, steps/rotation ▼ Stepper positions (steps)
