# TeenAstro Release Notes since version 1.5

## English

This update series focuses on practical improvements for everyday observing: better reliability, smoother workflows, and easier updates.

### User benefits

- More reliable GOTO, tracking, and alignment in real sessions.
- Better SHC behavior during slews and alignment phases.
- Stronger ASCOM support (Telescope + Focuser) with improved connectivity and compatibility.
- Better app experience on both Windows and Android (planetarium, catalog usage, coordinate handling, and connection stability).
- More robust and safer WiFi/Web behavior.
- Simpler update path with consolidated release artifacts.
